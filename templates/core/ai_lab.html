📐 Architecture Guide

🧱
Deep MLP
Classic multi-layer perceptron with He initialisation, LayerNorm, Dropout, and configurable activation (ReLU / LeakyReLU / ELU). Best for quick experimentation.
🔗
ResNet
Residual network with skip-connections. Each block learns a small Δ-correction instead of the full mapping, eliminating vanishing gradients in very deep stacks. Default choice.
👁️
AttentionNet
Transformer-inspired. Treats each input feature as a "token" and learns cross-feature dependencies via scaled dot-product attention. Attention weights are directly inspectable.
